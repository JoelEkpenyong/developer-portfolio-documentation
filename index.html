HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
Create a .row and add the appropriate number of .col-* columns.
  
<div class="row">
        
       { Level 1 column }

        <div class="row">
            <div class="col-6"> { Level 2 } </div>
            <div class="col-6"> { Level 2 } </div>
        </div>

</div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

JavaScript

jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
For more information, please visit http://www.jquery.com/
 
All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
<!-- Main Jquery JS -->
<script src="./assets/js/jQuery/jquery-3.5.1.slim.min.js" type="text/javascript"></script>        
<!-- Popper JS -->
<script src="./assets/js/popper/popper.min.js" type="text/javascript"></script> 
<!-- Bootstrap JS -->
<script src="./assets/js/bootstrap/bootstrap.min.js" type="text/javascript"></script>       
<!-- Swiper JS -->
<script src="./assets/js/swiper/swiper-bundle.min.js" type="text/javascript"></script>    
<!-- GSAP JS -->
<script src="./assets/js/gsap/gsap.min.js" type="text/javascript"></script>   
<!-- GSAP TextPlugin JS -->
<script src="./assets/js/gsap/TextPlugin.min.js" type="text/javascript"></script>   
<!-- GSAP ScrollTriger JS -->
<script src="./assets/js/gsap/ScrollTrigger.min.js" type="text/javascript"></script>   
<!-- Custom JS -->
<script src="./assets/js/custom js/app.js"></script>

 

Fonts

To change the embedded font, please take a look in the head tag of index.html:
 

 

To change the font, first go to http://www.google.com/webfonts choose a font and add the generated code to the top of global.css.
 
Then edit font-family in
  
              *{
                box-sizing: border-box;

                color: #fff;
            }

Sliders

Swiper.js is the slider you will find in the content. It's an essay to use lighweight carousel with some decent features.
 
testimonial Section slider has a div with the class 'slider-container' and an additional required wrapper, a div with the class 'slider-wrapper' each slide has '<div class="swiper-slide">' wrapper for easy find where swiper.js html code starts. If you wont remove some slide just remove div with all content like this '<div class="swiper-slide">'
  
<!-- Slides -->
<div class="swiper-slide">
    <div class="card d-flex flex-column justify-content-center align-items-center mx-2 mx-lg-4">
      <img src="./assets/imgs/placeholders/100x100.png" alt="" class="mt-n5 mt-md-n5" style="border-radius: 50%;" >
      <div class="card-body">
        <p class="card-txt">Some quick example text to build on the card title and make up the bulk of the card's content."/p>
        <h5 class="card-title text_primary text_large">Card Title"/p>
        <h6 class="card-subtitle mb-2">Card subtitle"/p>
    </div>          
</div>
 
If you want change option for sliders you need to open assets/js/custom js/app.js and find '// Initialize swiper.js' function and set true/false for option what you wont to change
 
  
// initialize swiper.js
var mySwiper = new Swiper('.swiper-container', {
    // Optional parameters
    loop: true,
    slidesPerView: 1,
    height: 500,

    // If we need pagination
    pagination: {
        el: '.swiper-pagination',
        dynamicBullets: true, 
    },

    // breakpoints
    breakpoints: {
        576.98: {
            slidesPerView:3,
        }
    },
})

You can find more documentation here https://swiperjs.com
